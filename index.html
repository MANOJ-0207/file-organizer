A smart way to organize your files with ease.

⬇️ Download for Windows (.exe)
📁 User Guide
💻 Developer Setup

🚀 Features Preview UI

  • Select folder to organize
  • Advanced renaming using date/time patterns
  • Filter files by type, name pattern (SQL-like), and date modified
  • Undo operations with a click
  • Supports reusable templates
  • Light/Dark theme switcher

📋 Notes Before You Start

  • No Node.js or Java installation needed – Amazon Corretto JRE is embedded.
  • After clicking "Commit", changes are permanent and can't be reverted.
  • Check the target directory once before confirming rename/move operations.

🧾 Field Instructions

  • Folder Path: Select the folder you want to organize.
  • File Types: Filter by extensions (.jpg, .pdf, etc).
  • Name Pattern: SQL-like wildcard, e.g. _data%.csv
  • Date Range: Filter files by last modified date.
  • Rename Pattern: Use tokens like {date}, {time}Example: IMG_{date}_{id}
  • Replace Pattern: Example: IMG → Event
    Example: IMG_123 → Event_123
  • Subfolder: If enabled, files will be moved to a subfolder.

🧱 Architecture Overview

+----------------------+      HTTP      +------------------------+
| Electron Frontend    |  <--------->   | Spring Boot Backend    |
| HTML/CSS/JS          |                | Java + REST APIs       |
+----------------------+                +------------------------+
  • Frontend: Electron (Node.js + Chromium)
  • Backend: Spring Boot with REST APIs
  • JAR runs with embedded Amazon Corretto JRE

🛠 Build Instructions

git clone https://github.com/MANOJ-0207/file-organizer.git
cd file-organizer

# Build Spring Boot
cd Backend
mvn clean install
mv target/fileOrganizer-backend.jar ../Backend/

# Electron
cd ..
npm install
npm start

📦 Packaging

npm run dist

🧠 Internals

  • Strategy Pattern: RenameStrategy and SortStrategy are interchangeable.
  • Decorator Pattern: Filter chains wrap around base logic.
  • SQL-like Pattern: Used for name-based file filtering.
  • Renaming and sorting logic is separated for reusability.
